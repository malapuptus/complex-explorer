# Bug Repro Pack (Internal)

Use this when a bug report is missing details or is intermittent.

## Required

- Repro steps (numbered) OR "could not reproduce"
- Expected vs actual
- Screenshot/video if UI-related
- Console errors (copy/paste)
- Environment (browser + OS)

## If state/data is involved

- If export exists: request an exported session file
- If import exists: confirm import reproduces the issue

## Classification

- **Severity:** P0 / P1 / P2
- **Type:** crash | wrong result | UI dead-end | performance | data loss | other
- **Repro rate:** 100% | intermittent | unknown

## When you can't repro

Ask reporter to try again with:

- A screen recording
- Console open during repro
- Exact input values / exact clicks
